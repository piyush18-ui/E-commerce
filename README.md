# E-commerce


A beginner-friendly full-stack commerce website built using
HTML, CSS, JavaScript, Node.js, and Express.

## Features
- Product listing
- Add to cart
- Remove from cart
- Frontend & Backend connected using API

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: JSON file

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `frontend/index.html` in browser
